Ce este Simple Linux?

Simple Linux este un asistent vizual care te ajută să gestionezi serviciile sistemului tău Linux într-un mod simplu și intuitiv.

Cum folosești aplicația?

Scrie numele unei comenzi sau al unui serviciu în câmpul principal și apasă Enter pentru a vedea pagina de manual. Apasă butonul Services Dashboard pentru a gestiona serviciile.

Services Dashboard

Vizualizează toate serviciile systemd, caută după nume, și accesează detalii precum status, PID și comenzi rapide.

Man Page

Citește documentația oficială pentru orice comandă Linux. Selectează text și apasă Ctrl+T sau butonul Translate pentru a traduce cu DeepL.

Settings

Adaugă cheia ta API DeepL și alege limba în care vrei să fie tradus textul din paginile de manual.
